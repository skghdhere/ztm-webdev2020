Currying ðŸŒŽ Learn More 

Currying is a transformation of functions that 
translates a function from callable as f(a, b, c)
into callable as f(a)(b)(c).


const multiply = (a,b) => a*b;
const curriedMultiply = (a) => (b) => a*b;

curriedMultiply(3)    // returns  (b) => a*b
curriedMultiply(3)(4) // returns 12

This makes it more extensible

const multiplyBy5 = curriedMultiply(5)
multiplyBy5(9) // return 45

It is the act of putting two functions together

const compose = (f,g) => (a) => f(g(a));
