of two types Global or Local - within {}.

Context is most often determined by how a function is invoked.
When a function is called as a method of an object,

classes OOP|üåè MDN Docs 

Classes are in fact "special functions", and just as you
can define function expressions and function 
declarations, the class syntax has two components:
class expressions and class declarations.


  // Class declarations
class Player {
  constructor(name, type) {
    this.name = name;
    this.type = type;
  }
  // Introducer
  introduce() {
    console.log(`Hi I am ${this.name}, a ${this.type}.`);
  }
}
class Wizard extends Player {
  constructor(name, type){
    super(name, type)
    }
    play() {
      console.log(`Heyyy i', a ${this.type}`)
    }
}

const wizard1 = new Wizard('Shelly', 'Healer');
const wizard1 = new Wizard('Shawn', 'Dark Magician');
